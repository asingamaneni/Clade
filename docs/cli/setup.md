# clade setup

Interactive setup wizard for first-time configuration.

## Usage

```bash
clade setup
```

## What It Does

1. Creates the `~/.clade/` directory structure
2. Detects the installed Claude CLI version
3. Walks you through channel configuration (Slack, Telegram, Discord)
4. Sets up the initial `config.json`
5. Verifies the setup with a health check
